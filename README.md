# Twitter Clone Finatra

## Requirements

- sbt
- MySQL 5.7
- JDK 1.8

install yourself.

## Library dependencies

- finatra
- Quill
- slf4j Logger
- Spring Security

They are defined in `build.sbt`

## Setup

### Download code

```
$ git clone https://github.com/kijimaru/twitter-clone-finatra.git
```

### 

### Boot application

```
$ cd twitter-clone-fintra
$ chmod -x boot.sh
$ ./boot.sh
```
